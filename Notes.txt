1. PROJECT DESCRIPTION
Why Did My Model Fail? is an AI-powered web application that helps users identify why machine learning models perform poorly. It analyzes datasets, trains multiple ML models, detects issues like overfitting or class imbalance, and provides improvement suggestions.

2. REQUIREMENTS
- Python 3.8+
- Flask
- Pandas
- NumPy
- Scikit-learn
- ReportLab
- Modern browser (Chrome/Edge/Firefox)

3. HOW TO RUN PROJECT
1. Download project files
2. Install dependencies:
   pip install -r requirements.txt
3. Run project:
   python app.py
4. Open browser:
   http://localhost:5000

4. HOW IT WORKS (TECH FLOW)
Dataset Upload → Data Cleaning → Preprocessing → Train 7 ML Models → Performance Evaluation → Failure Detection → Insights & PDF Report

5. FEATURES
- Supports CSV, Excel, JSON
- Automatic preprocessing
- Trains 7 ML models
- Detects overfitting/underfitting
- Feature importance analysis
- Interactive charts
- PDF report generation

6. COMMON ISSUES & FIXES
- Module error → Reinstall dependencies
- Port busy → Change Flask port
- Upload failed → Check file format/size
- Slow analysis → Use smaller dataset

7. SUPPORTED BROWSERS
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari

8. TIPS FOR BEST RESULTS
- Use clean datasets
- Keep target column last
- Remove duplicate rows
- Use 100+ samples for better accuracy
