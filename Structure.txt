================================================================================
                        SUPPORTED FILE FORMATS
================================================================================

Data File Input Formats:
  • CSV (.csv) - Comma-separated values
  • TSV (.tsv) - Tab-separated values
  • TXT (.txt) - Text files (auto-delimiter detection)
  • Excel (.xlsx, .xls, .xlsm, .xlsb, .ods)
  • JSON (.json) - JavaScript Object Notation
  • Parquet (.parquet) - Apache Parquet format
  • Feather (.feather) - Arrow Feather format
  • Pickle (.pkl, .pickle) - Python serialized objects
  • HDF5 (.h5, .hdf5) - Hierarchical Data Format
  • Stata (.dta) - Stata statistics files
  • SAS (.sas7bdat) - SAS database files
  • SPSS (.sav) - SPSS statistics files

Output Formats:
  • HTML (Web interface visualization)
  • PDF (Generated report with findings)
  • JSON (API response data)

================================================================================
                         MODELS INCLUDED
================================================================================

1. Decision Tree Classifier
   - Default depth: 8
   - Good for interpretability

2. Random Forest Classifier
   - 50 estimators
   - Parallel processing enabled

3. Logistic Regression
   - Max iterations: 300
   - Solver: SAGA

4. Support Vector Machine (SVM)
   - Kernel: Linear
   - Fast and efficient

5. K-Nearest Neighbors
   - Parallel processing enabled
   - Adaptive k selection

6. Naive Bayes (Gaussian)
   - Probabilistic classifier
   - Baseline model

7. Gradient Boosting Classifier
   - 50 estimators
   - Max depth: 4
   - Subsample: 0.8

================================================================================
                       ANALYSIS FEATURES
================================================================================

1. Data Profiling:
   - Column statistics (min, max, mean, std)
   - Data type detection
   - Missing value analysis
   - Unique value counts
   - Distribution histograms

2. Model Training:
   - Parallel training of 7 models
   - Automatic best model selection
   - Cross-validation scoring
   - Performance metrics on test set

3. Failure Diagnosis:
   - Overfitting detection
   - Underfitting detection
   - Class imbalance analysis
   - Feature variance assessment
   - Correlation analysis

4. Performance Analysis:
   - Accuracy, Precision, Recall, F1-score
   - Confusion matrix
   - ROC-AUC curve (binary classification)
   - Per-class metrics
   - Learning curves

5. Feature Analysis:
   - Feature importance scores
   - Permutation importance
   - Correlation matrix
   - Variance detection
   - Skewness analysis

6. Hyperparameter Tuning:
   - Automated RandomizedSearchCV
   - Performance improvement tracking
   - Best parameters recommendation

7. Report Generation:
   - PDF report with findings
   - Diagnostic summary
   - Recommendations for improvement
